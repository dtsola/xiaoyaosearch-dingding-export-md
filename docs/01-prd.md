# PRD：小遥搜索钉钉导出工具 MVP版本

## 1. 产品定位（一句话）

为 **钉钉文档用户** 提供 **设计精美、体验流畅** 的 **浏览器插件文档导出工具**

---

## 2. 用户故事（核心场景）

### 场景1：知识管理爱好者 - 个人文档备份

- **作为** 知识管理爱好者
- **我想要** 将我的钉钉文档批量导出到本地
- **以便于** 防止云端数据丢失，并在离线状态下也能查看我的文档

### 场景2：自由职业者 - 迁移到本地笔记工具

- **作为** 自由职业者
- **我想要** 将钉钉团队知识库导出为 Markdown 格式
- **以便于** 迁移到 Obsidian/Logseq 等本地知识管理工具

### 场景3：创业团队 IT 运维 - 定期备份归档

- **作为** 创业团队 IT 运维
- **我想要** 一键导出完整的知识库
- **以便于** 定期备份企业知识资产，满足合规归档要求

### 场景4：技术文档维护者 - 版本控制

- **作为** 技术文档维护者
- **我想要** 导出 API 文档并保持目录结构
- **以便于** 提交到 Git 仓库进行版本控制和协作

### 场景5：学生用户 - 跨平台使用

- **作为** 经常更换设备的学生
- **我想要** 在不同浏览器上都能使用导出功能
- **以便于** 在 Windows/macOS 间灵活切换学习资料

---

## 3. 功能清单（MVP范围）

### P0功能（必须有）

#### 3.1 文档导出功能
- [ ] **单个文档导出**
  - 支持钉钉文档（.adoc）导出为 **.md / .docx / .pdf**
  - 支持钉钉表格（.axls）导出为 **.xlsx**
  - 支持钉钉白板（.adraw）导出为 **.jpg**
  - 支持钉钉脑图（.amind）导出为 **.jpg**
  - 自动下载文档中的图片和附件

#### 3.2 批量导出功能
- [ ] **可视化文档树**
  - 侧边栏面板展示文档/文件夹层级结构
  - 支持勾选单个文档
  - 勾选文件夹时自动勾选所有子项
  - 支持全选/取消全选

- [ ] **批量导出操作**
  - 点击"导出选中"按钮开始批量下载
  - 保持原有目录结构
  - 实时显示导出进度（已导出/总数）
  - 显示每个文件的导出状态（等待中/导出中/成功/失败）

#### 3.3 知识库导出功能
- [ ] **完整知识库导出**
  - 递归获取知识库所有文档
  - 保持知识库的层级结构
  - 支持选择知识库根节点进行导出

- [ ] **分页加载**
  - 大型知识库支持分页加载文档列表
  - 自动加载更多文档（滚动到底部或点击"加载更多"）

#### 3.4 配置管理功能
- [ ] **导出格式配置**
  - 设置文档默认导出格式（.md / .docx / .pdf）
  - 设置表格默认导出格式（.xlsx）
  - 设置白板默认导出格式（.jpg）
  - 设置脑图默认导出格式（.jpg）

- [ ] **配置存储**
  - 使用浏览器 LocalStorage 保存用户配置
  - 下次打开自动恢复上次的配置

#### 3.5 用户界面
- [ ] **侧边栏面板**
  - 在钉钉文档页面右侧滑出
  - 苹果风格设计（简洁、优雅、现代化）
  - 支持展开/收起
  - 响应式布局，适配不同屏幕尺寸

- [ ] **视觉反馈**
  - 加载状态指示（Loading 动画）
  - 成功/失败的明确提示
  - 鼠标悬停效果
  - 按钮点击反馈

#### 3.6 错误处理
- [ ] **错误提示**
  - 网络错误提示
  - 权限不足提示
  - 文档不存在提示
  - 导出失败提示

- [ ] **重试机制**
  - 单个文件失败支持重新导出
  - 批量导出失败支持重试失败的文件

### P1功能（最好有）

- [ ] **导出历史记录**
  - 记录最近的导出操作
  - 支持快速重新导出

- [ ] **快捷键支持**
  - 快捷键打开/关闭侧边栏
  - 快捷键开始导出

- [ ] **深色模式**
  - 跟随钉钉主题自动切换
  - 手动切换深色/浅色模式

### P2功能（未来考虑）

- [ ] **多语言支持**
  - 中文/英文界面切换

- [ ] **导出模板**
  - 自定义 Markdown 导出模板
  - 自定义文件命名规则

- [ ] **增量导出**
  - 仅导出有更新的文档
  - 对比上次导出时间戳

## 7. 非功能需求

### 7.1 性能要求

| 指标 | 要求 |
|------|------|
| **侧边栏打开速度** | < 500ms |
| **文档树渲染** | 支持 1000+ 文档节点不卡顿 |
| **单个文件导出** | < 10秒（取决于文件大小） |
| **内存占用** | < 100MB |

### 7.2 兼容性要求

| 浏览器 | 版本要求 | 优先级 |
|--------|---------|--------|
| **Chrome** | 90+ | P0 |
| **Edge** | 90+ | P0 |
| **Firefox** | 88+ | P1 |
| **Safari** | 15+ | P2 |

### 7.3 安全要求

| 项目 | 要求 |
|------|------|
| **权限请求** | 仅请求最小必要权限（activeTab, storage） |
| **数据传输** | 使用钉钉页面已有 Cookie，不额外存储敏感信息 |
| **代码安全** | Content Security Policy (CSP) 防护 |
| **开源透明** | 代码完全开源，接受社区审计 |

### 7.4 可用性要求

| 指标 | 要求 |
|------|------|
| **界面语言** | 中文（简体） |
| **操作引导** | 首次使用显示引导页 |
| **错误提示** | 所有错误都有明确的中文提示 |
| **操作反馈** | 每个操作都有视觉或状态反馈 |

### 7.5 可维护性要求

| 项目 | 要求 |
|------|------|
| **技术栈** | Vue 3 / React + TypeScript |
| **代码规范** | ESLint + Prettier |
| **组件化** | 高度模块化，便于复用和测试 |
| **文档** | 完整的开发文档和用户文档 |

## 9. 设计参考

### 9.1 苹果风格设计原则

- **简洁** - 去除不必要的装饰，专注内容
- **优雅** - 使用流畅的动画和过渡效果
- **一致** - 统一的设计语言和交互模式
- **层次** - 清晰的信息层级和视觉引导

### 9.2 色彩方案

| 用途 | 颜色 | Hex |
|------|------|-----|
| **主色调** | 蓝色 | #007AFF |
| **成功** | 绿色 | #34C759 |
| **警告** | 橙色 | #FF9500 |
| **错误** | 红色 | #FF3B30 |
| **背景** | 浅灰 | #F5F5F7 |
| **卡片背景** | 白色 | #FFFFFF |

- 中文：系统默认无衬线字体
- 英文/数字：SF Pro / -apple-system

## 10. 附录

### 10.1 钉钉文档类型说明

| 后缀 | 类型 | 说明 |
|------|------|------|
| `.adoc` | 钉钉文档 | 支持富文本、图片、表格等 |
| `.axls` | 钉钉表格 | 类似 Excel 的在线表格 |
| `.adraw` | 钉钉白板 | 在线协作白板 |
| `.amind` | 钉钉脑图 | 思维导图 |

### 10.2 导出格式说明

| 格式 | 优势 | 适用场景 |
|------|------|---------|
| **.md** | 纯文本，易于版本控制，可迁移到任何笔记工具 | 技术文档、知识库 |
| **.docx** | 保留完整格式，兼容 Office | 商务文档、正式文档 |
| **.pdf** | 固定格式，适合打印和分享 | 归档、对外分享 |
| **.xlsx** | Excel 原生格式 | 数据表格 |
| **.jpg** | 通用图片格式 | 白板、脑图可视化 |
