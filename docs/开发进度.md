#### 🎯 任务 2.5：复刻钉钉API封装

**参考文件**：[olds/ding-doc-downloader/src/script/api.js](../olds/ding-doc-downloader/src/script/api.js) (约870行)

**待实现的API函数**：

| 函数名 | 功能 | 实现状态 | 优先级 |
|--------|------|---------|--------|
| `getDocList(dentryUuid)` | 获取文档列表（支持分页） | ⬜ 未开始 | P0 |
| `getSpaceInfo(spaceId)` | 获取空间信息 | ⬜ 未开始 | P0 |
| `downloadAdoc(id, format)` | 导出文档为 .md/.docx/.pdf | ⬜ 未开始 | P0 |
| `downloadAxls(id)` | 导出表格为 .xlsx | ⬜ 未开始 | P0 |
| `downloadBoard(id)` | 导出白板为 .jpg | ⬜ 未开始 | P0 |
| `downloadMind(id)` | 导出脑图为 .jpg | ⬜ 未开始 | P0 |
| `downloadDocument(id)` | 下载原始文件 | ⬜ 未开始 | P0 |

**技术要点**：
- 钉钉 GraphQL API 调用
- 导出流程：upload_info → submitExportJob → queryExportStatus
- Cookie 和 CSRF Token 处理
- 轮询机制实现
