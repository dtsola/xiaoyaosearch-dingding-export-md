### 1.2 正文结构

#### 背景故事

作为一名独立开发者，我之前开发了 [小遥搜索](https://github.com/dtsola/xiaoyaosearch)——一个本地 AI 搜索工具，目前已经获得了 1000+ GitHub Stars。

> **小遥搜索：听懂你的话、看懂你的图，用 AI 找到本地任何文件。让搜索像聊天一样简单。**

前几天，一个朋友突然找到我，声音都带着哭腔：

"哥们，我被坑了！离职要交接工作，公司飞书里有 500 多篇文档，HR 说必须导出才能走。可飞书根本不支持批量导出啊！"

我让他试试飞书官方的导出功能。结果发现，只能一篇一篇手动操作，而且导出的格式需要额外处理才能用。

他试着导出了几十篇，整个人都要崩溃了："这要导到什么时候啊？离职 deadline 就在后天！"

调研了一圈，发现市面上确实没有好用的飞书导出工具。

**那就自己写一个吧！**

作为独立开发者，遇到问题就解决问题，这是我的日常。

#### 解决方案

所以我开发了这个工具——一个基于 TypeScript + Node.js 的命令行工具。

核心功能：
✅ 单个文档/批量文档导出
✅ 文件夹递归导出
✅ 完整知识库导出
✅ 自动下载图片和附件
✅ 增量导出（只导出有更新的文档）
✅ 跨平台支持（Windows/macOS/Linux）

一行命令即可完成导出：

```bash
# 导出单个文档
feishu-export export -d doxcnXXXXXXXX

# 批量导出
feishu-export docs --file docs.txt

# 导出知识库
feishu-export wiki wiki_id
```

安装使用超简单：
```bash
npm install -g xiaoyaosearch-feishu-export
feishu-export init
```
