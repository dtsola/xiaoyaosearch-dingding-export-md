## 1. 测试环境准备

### 1.1 浏览器要求

- [ ] Chrome 版本 ≥ 114（支持 Manifest V3 和 Side Panel API）
- [ ] Edge 版本 ≥ 114（Chromium 内核）

### 1.2 加载扩展

```bash
# 1. 构建扩展
npm run build

# 2. 打开 Chrome 扩展管理页面
# 访问 chrome://extensions/

# 3. 开启开发者模式
# 点击右上角"开发者模式"开关

# 4. 加载扩展
# 点击"加载已解压的扩展程序"
# 选择项目根目录下的 dist 文件夹

# 5. 确认扩展已加载
# 扩展列表中应出现"小遥搜索钉钉导出工具"
```

**验证点：**
- [ ] 扩展成功加载
- [ ] 无控制台错误信息
- [ ] 扩展图标显示正常（如已配置）
