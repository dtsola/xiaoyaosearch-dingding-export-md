**HtmlParser.ts** - HTML 解析器
```typescript
import * as cheerio from 'cheerio';

export class HtmlParser {
  /**
   * 从 HTML 中提取 dentryKey
   */
  static extractDentryKey(html: string): string {
    const $ = cheerio.load(html);
    const scriptContent = $('#mainsite_server_content').html();
    if (!scriptContent) {
      throw new Error('无法找到 mainsite_server_content');
    }
    const data = JSON.parse(scriptContent);
    return data.dentryKey;
  }
}
```

### 4.2 document.ts - 文档类型

```typescript
export interface DocumentData {
  documentContent: {
    checkpoint: {
      content: string; // 加密的 JSON 字符串
    };
  };
  title: string;
  revisionId: string;
}

export interface ContentNode {
  type: string;
  content?: any;
  children?: ContentNode[];
}

export interface ParsedDocument {
  title: string;
  content: DocumentContent[];
  images: Image[];
  assets: Asset[];
  metadata: DocumentMetadata;
}

export interface DocumentContent {
  type: 'paragraph' | 'heading' | 'code' | 'table' | 'list' | 'image';
  level?: number; // for heading
  text?: string;
  language?: string; // for code
  rows?: string[][]; // for table
}

export interface DocumentMetadata {
  id: string;
  title: string;
  createdAt: string;
  updatedAt: string;
  author?: string;
}
```

## 8. 代码规范

### 8.1 命名约定

| 类型 | 命名风格 | 示例 |
|------|---------|------|
| 类 | PascalCase | `ConfigManager`, `DingTalkAPI` |
| 接口 | PascalCase，无 I 前缀 | `Config`, `DocumentData` |
| 类型别名 | PascalCase | `ExportOptions`, `ParsedDocument` |
| 枚举 | PascalCase | `ErrorCode` |
| 常量 | UPPER_SNAKE_CASE | `API_CONFIG`, `USER_AGENT` |
| 函数/方法 | camelCase | `loadConfig`, `exportDocument` |
| 私有方法 | camelCase，可选 _ 前缀 | `_saveCookie`, `loadCookie` |
| 变量 | camelCase | `docId`, `outputDir` |
| 文件名 | PascalCase（类文件）<br>camelCase（工具文件） | `ConfigManager.ts`<br>`file.ts`, `logger.ts` |
| 目录名 | kebab-case 或 camelCase | `api/`, `generators/`, `utils/` |

### 8.2 文件组织规范

- **一个文件一个类/主要导出**
- **相关类型定义在同一文件**
- **工具函数按功能分组到不同文件**
- **测试文件与源文件对应，放在 tests/ 目录下**

### 8.3 代码风格

- 使用 **2 空格缩进**
- 使用 **单引号**（JavaScript/TypeScript）
- 使用 **分号**
- **最大行宽 120 字符**
- **函数间空一行**
- **逻辑块间空两行**
