### **3.3 HTTP 请求封装（script/Http.js）**

```javascript
export default function httpRequest(options) {
    return new Promise((resolve, reject) => {
        const xhr = new XMLHttpRequest();

        // 请求头设置
        if (options.headers) {
            Object.keys(options.headers).forEach(key => {
                xhr.setRequestHeader(key, options.headers[key]);
            });
        }

        // 进度监控
        if (options.onProgress) {
            xhr.addEventListener('progress', (e) => {
                options.onProgress(e.loaded, e.total);
            });
        }

        // 响应处理
        xhr.addEventListener('load', () => {
            if (options.onEnd) {
                options.onEnd(null, xhr.response);
            }
            resolve(xhr.response);
        });

        // 错误处理
        xhr.addEventListener('error', () => {
            if (options.onEnd) {
                options.onEnd(new Error('Network error'), null);
            }
            reject(new Error('Network error'));
        });

        xhr.open(options.method || 'GET', options.url);
        xhr.send(options.data);
    });
}
```

## 5. 扩展架构

### **5.1 消息通信流程**

```
┌─────────────────────────────────────────────────────────────┐
│                         用户浏览器                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────────┐      sendMessage      ┌──────────────┐   │
│  │background.js │ ─────────────────────► │ content.js   │   │
│  │ (Service     │                         │              │   │
│  │  Worker)     │ ◄───────────────────── │  初始化 Baby │   │
│  └──────────────┘      onMessage         └──────────────┘   │
│         ▲                                               │    │
│         │ chrome.action.onClicked                         │    │
│         │                                               │    │
│  ┌──────┴──────┐                                         │    │
│  │ 用户点击图标  │                                         │    │
│  └─────────────┘                                         │    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### **5.2 代码执行顺序**

1. **用户访问钉钉文档页面**
   - `content.js` 自动注入（通过 manifest.json 配置）

2. **用户点击扩展图标**
   - `background.js` 监听 `chrome.action.onClicked` 事件

3. **background.js 发送消息**
   ```javascript
   chrome.tabs.sendMessage(tabId, {action: 'showDownloader'});
   ```

4. **content.js 接收消息**
   ```javascript
   chrome.runtime.onMessage.addListener((request, sender, sendResponse) => {
       if (request.action === 'showDownloader') {
           initApp(); // 调用 app.js 的初始化函数
       }
   });
   ```

5. **app.js 初始化应用**
   - 创建 Baby 实例
   - 渲染 Main 组件
   - 加载文档列表

### **命名约定**

| 类型 | 命名风格 | 示例 |
|------|---------|------|
| 组件类 | PascalCase | `class MainComponent` |
