## 苹果风格设计方案

### 设计原则
- ✨ **简洁至上**：去除多余装饰
- 🍏 **毛玻璃效果**：半透明 + 背景模糊
- 🎨 **浅色调**：白色 + 浅灰 + 蓝色点缀
- 🔵 **SF Pro 字体**：系统字体栈优先
- 📐 **大圆角**：12-16px 圆角
- 🌫 **柔和阴影**：扩散性阴影，无硬边

## 核心改动点

### 1. 颜色系统

| 用途 | DaisyUI | 苹果风格 |
|------|---------|----------|
| 主背景 | `bg-base-100` | `rgba(255,255,255,0.8)` |
| 卡片背景 | `bg-base-100` | `rgba(255,255,255,0.95)` |
| 主色调 | `dddd-btn-primary` | `#007AFF` |
| 警告色 | `dddd-alert-warning` | `#FF9500` |
| 边框色 | `border-zinc-300` | 透明 |

### 2. 组件对比

#### 按钮
```javascript
// 当前 - DaisyUI
class: "dddd-btn dddd-btn-primary text-white"

// 苹果风格
class: "apple-button apple-button-primary"
```

**样式差异：**
| 属性 | DaisyUI | 苹果风格 |
|------|---------|----------|
| 圆角 | 4px | 12px |
| 阴影 | `shadow-md` | `box-shadow: 0 2px 8px rgba(0,0,0,0.1)` |
| 过渡 | 0.2s | 0.3s cubic-bezier(0.25, 0.1, 0.25, 1) |

#### 复选框
```javascript
// 当前
<input class="dddd-checkbox dddd-checkbox-xs" />

// 苹果风格
<div class="apple-checkbox">
  <input type="checkbox" />
  <svg class="checkmark">...</svg>
</div>
```

**特点：**
- 更大的点击区域
- 自定义勾选动画
- SF Symbols 图标

#### 进度条
```javascript
// 当前
<div class="dddd-radial-progress" style="--value:50">50%</div>

// 苹果风格
<div class="apple-progress">
  <div class="progress-bar" style="width:50%"></div>
  <span class="progress-text">50%</span>
</div>
```

## 实现方案

### 方案 A：完全替换 DaisyUI（推荐）

**优点：**
- 完全控制设计
- 文件更小
- 性能更好

**缺点：**
- 工作量大
- 需要重写所有组件

**时间：** 2-3 天

---

### 方案 B：DaisyUI + 自定义覆盖（快速）

**优点：**
- 改动小
- 快速见效

**缺点：**
- 仍有 DaisyUI 基础样式
- 文件较大

**时间：** 0.5-1 天

---

## 推荐方案：方案 A 渐进式改造

### 阶段划分

| 阶段 | 内容 | 时间 |
|------|------|------|
| **1. 基础样式** | 颜色、字体、圆角、阴影 | 0.5 天 |
| **2. 毛玻璃效果** | 背景模糊、半透明 | 0.5 天 |
| **3. 组件重写** | 按钮、复选框、进度条 | 1 天 |
| **4. 动画优化** | 过渡效果、加载动画 | 0.5 天 |

### 技术栈

```
保持：Tailwind CSS + Baby 框架
移除：DaisyUI
新增：自定义苹果风格组件库
```
